## Rime 配置文件

### 使用

1. 下载安装 [RIME](https://rime.im/)
2. 点击菜单栏输入法图标，选择 `用户文件夹` ，将 `config` 中的文件粘贴进去，并选择覆盖
3. 点击菜单栏输入法图标，选择 `重新部署`
