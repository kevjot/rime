## Rime Settings (multilingual version)

### Features

- 中文
  - 支持全拼、小鹤双拼、小鹤音形输入方案
  - Emoji、中英混输
  - 两分、笔画反查
- 英语
- 日语
- 西班牙语
- 韩语
- 德语
- 俄语
- 阿拉伯语

### Usage

1. 下载安装 [RIME](https://rime.im/)
2. 点击菜单栏输入法图标，选择 `用户文件夹` ，将文件粘贴进去，选择覆盖
3. 点击菜单栏输入法图标，选择 `重新部署`
